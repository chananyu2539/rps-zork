.DS_Store :  is a file that stores custom attributes of its containing folder, created in any directory (folder) accessed by the Finder application. Thus, it doesn't deal with any of my application
 
*.iml : is a module file created by IntelliJ IDEA that stores information about a development module. Thus, it not neccessary for all users and doesn't deal with application development

target/ : is a folder created by maven after compile the project and cleaned after mvn clean command. Thus, doesn't need to be tracked of.

/.idea : folder created by Intellij to store all the project specific settings for project. Doesn't have to be keeping track of.
