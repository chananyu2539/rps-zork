Game consists of three level:
    1st floor - you need to defeat First floor boss to proceed to second floor
    2nd floor - you must find 2 keys and combine them to create portal to 3rd floor
    3rd floor - defeat final boss in 1-on-1 match and win the game

Here are the lists of commands you can use:
info => check what does this room contain.
go {dir} => moving to the dir direction where dir can be 'north','east','west','south'
take {potion} => to pick up potion, potion includes 'healingpotion','debuffpotion',can store atmost 6 items
use {potion} => use potion. potion includes 'healingpotion','debuffpotion'
attack => engage in battle if there is the guardian in the room
rock => play rock, can be used when in battle
paper => play paper, can be used when in battle
scissors => play scissors, can be used when in battle
drop {i} => drop item in the i'th slot
flee => to escape from the current room the the last visited one
position => check the current postion of the character
checkbag => check all item inside your bag
checkhp => check current HP of the player
combine => combine quest key to move to final boss room
quit => quit the game :(


===================================================================================================================== 

When there is a Guardian in the room, you cannot go to other rooms unless it's dead
If you decide to flee, you will lose 5 HP.
If you win, enemy lose 10 HP. If you lose, you lose 5 hp.
Healing potion can heal a total of 10 HP per potion
Debuff potion will make Guardian impossible to play rock for 1 round
You can stored only upto 6 potions
